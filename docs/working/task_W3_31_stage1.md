### "연령대+성별 + 임산부" 가설 검증

| 검증 항목 | 결과 |

| **데이터 풍부함** | ✅ 5 카테고리 모두 5~7종 권장 영양제 명확 |

## 65세+ 시니어 추가 여부 분석

### 검색 결과 시사점
- 60대+ 노화 / 오메가3 / 비타민D / 항산화 — 40~60대와 일부 중복
- 70대+ 단백질 보충 / 비타민B12 흡수 저하 — 별도 카테고리 가치 있음

### 옵션
- **(가) V1 포함** — "65세+ 시니어" 6번째 카테고리 추가
- **(나) V1 미포함, V1.x 추가** — 일단 5 카테고리로 시작
- **(다) 40~60대를 40~70대로 확장** — 중복 회피

### 추천: **(나) V1.x로 미룸**

이유:
- 5 카테고리로 V1 본질 검증 가능
- 시니어 권장은 의료적 톤 인접 위험 ↑ (만성질환 / 약물 상호작용)
- V1 출시 후 사용자 데이터 보고 추가
