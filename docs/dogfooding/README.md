# dogfooding/ — 30일 도그푸딩 노트

> **사용 시점**: W5부터 (V1.0 출시 후 즉시 시작)

## 정의 (기획서 §30일 도그푸딩 정의)

> **도그푸딩(Dogfooding):** 메이커가 자기 앱을 매일 실생활에서 사용하는 것. 단순 테스트가 아니라 **"이 앱이 진짜 가치 있나?"** 검증.

### 30일이 의미 있는 이유
- **첫 주**: 신기해서 잘 씀 (가짜 신호)
- **둘째 주**: 익숙해짐
- **셋째 주**: 진짜 시험대 — 진짜 매일 쓰게 되나?
- **넷째 주**: 결론 — 일상에 자리 잡았나?

V1.0 출시 후 즉시 시작. 결과로 V1.1 방향 결정.

## 검증 명제 3개 (기획서 §검증 가능한 명제)

1. **명제 1**: "먹었나 헷갈렸던 경험"을 물으면 70% 이상이 "있다"고 답한다 → 외부 사용자 5명 인터뷰
2. **명제 2**: 명확한 시각적 증거를 본 후 "먹었나?" 의문이 다시 생기는 빈도가 줄어든다 → 30일 도그푸딩 자체 관찰
3. **명제 3**: 기록이 누적되어 보이면 영양제 복용 자체의 지속률이 올라간다 → 30일 사용 후 누락률 비교
