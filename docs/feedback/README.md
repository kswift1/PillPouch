# feedback/ — 작업지시자 피드백 (AI 작성 금지) ⛔

작업지시자가 코드 리뷰 / 단계 보고서 검토 후 작성하는 곳.

## 핵심 룰
**AI는 이 폴더에 글을 쓰지 않는다.** Claude/Codex 등 AI 에이전트가 자동으로 자기 작업에 대한 피드백을 쓰면 의미가 사라진다.

## 파일명
`task_W{N}_{이슈}_feedback.md`

## 형식 (작업지시자 자유)
- 잘한 점
- 고쳐야 할 점 (다음 PR/태스크에 반영)
- 패턴/원칙으로 박제할 것 → 추후 ADR 또는 CLAUDE.md 업데이트로 연결

## 사용 시점
- M 태스크: 최종보고서 검토 후
- L 태스크: 단계보고서마다 / 최종보고서 후
